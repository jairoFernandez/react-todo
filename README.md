Librerías usadas:

# Instrucciones instalación

### Desarrollo
```sh
  yarn install
  yarn start
```

### Producción 

Despliegue en github pages, se debe configurar la ruta en el package 

```json
{ 

  "homepage": "http://usuario.github.io/nombre-proyecto",

}
```

```sh
  yarn build
  yarn deploy
```
